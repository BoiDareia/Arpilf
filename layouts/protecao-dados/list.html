2. Dados Pessoais Recolhidos

A ARPILF recolhe dados pessoais exclusivamente através do formulário de contacto disponível no website. Os dados recolhidos são:

  • Nome (obrigatório)
  • Endereço de email (obrigatório)
  • Número de telefone (opcional)
  • Assunto da mensagem (obrigatório)
  • Conteúdo da mensagem (obrigatório)

Não são recolhidos dados sensíveis (dados de saúde, origem étnica, convicções políticas ou religiosas) através do website.

3. Finalidade do Tratamento

Os dados pessoais recolhidos são tratados exclusivamente para as seguintes finalidades:

  • Responder a pedidos de informação enviados através do formulário de contacto
  • Dar seguimento a questões ou solicitações apresentadas pelos visitantes
  • Contactar o remetente para esclarecimento ou resposta à sua mensagem

Os dados pessoais não são utilizados para fins de marketing, publicidade ou qualquer outra finalidade que não as acima indicadas.

4. Base Legal do Tratamento

O tratamento dos dados pessoais recolhidos através do formulário de contacto tem como base legal:

  • Consentimento do titular (artigo 6.º, n.º 1, alínea a) do RGPD) — o visitante consente expressamente o tratamento dos seus dados ao preencher e submeter o formulário de contacto, assinalando a checkbox de consentimento obrigatória.

O consentimento pode ser retirado a qualquer momento, sem que isso comprometa a licitude do tratamento efetuado com base no consentimento previamente dado.

5. Prazo de Conservação dos Dados

Os dados pessoais recolhidos através do formulário de contacto são conservados pelo período estritamente necessário para dar resposta à solicitação do visitante, e no máximo por 12 meses após a última interação.

Findo este prazo, os dados são eliminados de forma segura, salvo se existir obrigação legal de conservação por período superior.

6. Partilha de Dados com Terceiros

A ARPILF não partilha, vende ou cede dados pessoais a terceiros, exceto quando tal seja exigido por lei ou por ordem de autoridade competente.

Os dados recolhidos são tratados exclusivamente pela equipa da ARPILF responsável pela gestão das comunicações recebidas.

7. Medidas de Segurança

A ARPILF adota medidas técnicas e organizativas adequadas para proteger os dados pessoais contra o acesso não autorizado, a perda, a destruição ou a alteração acidental, nomeadamente:

  • Comunicação encriptada via HTTPS/SSL em todo o website
  • Validação e sanitização de todos os dados submetidos no formulário de contacto
  • Proteção contra spam através de mecanismos automáticos
  • Acesso restrito aos dados recolhidos, limitado aos colaboradores autorizados

8. Direitos dos Titulares dos Dados

Nos termos do RGPD, o titular dos dados pessoais tem os seguintes direitos:

  • Direito de acesso — Solicitar informação sobre os dados pessoais que a ARPILF detém sobre si.
  • Direito de retificação — Solicitar a correção de dados pessoais inexatos ou incompletos.
  • Direito ao apagamento — Solicitar a eliminação dos seus dados pessoais.
  • Direito à limitação do tratamento — Solicitar a restrição do tratamento dos seus dados pessoais.
  • Direito à portabilidade — Solicitar a transferência dos seus dados pessoais para outra entidade.
  • Direito de oposição — Opor-se ao tratamento dos seus dados pessoais a qualquer momento.
  • Direito de retirar o consentimento — Retirar o consentimento dado, sem afetar a licitude do tratamento anterior.
